# Student Management System

A C-based Student Management System with a graphical user interface using Raylib. This system allows efficient management of student records, courses, and assignments.

## Features
- Add, edit, and view student records
- Manage course information
- Track assignments and grades
- User-friendly graphical interface using Raylib
- Data stored locally for offline use

## Technologies Used
- C 
- Raylib for GUI
- Windows environment (Linux/Mac compatible with Raylib setup)
